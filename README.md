# Early-Detection-of-Cognitive-Decline-using-Multimodal-Data

A cognitive screening platform designed to evaluate multiple cognitive domains through three scientifically inspired tasks: Numerical Ability, Stroop Test, and Mental Rotation Task. The system combines performance accuracy, reaction time, attention control, processing speed, and spatial reasoning metrics into a unified cognitive assessment pipeline.

Using dataset-driven normalization, Composite Cognitive Score (CCS) generation, age-adjusted benchmarking, and machine learning–based risk estimation, the platform provides an interpretable cognitive risk profile categorized into Low Risk, Mild Risk, or High Risk. The application is built with Streamlit and is intended for academic research, cognitive analytics, and early screening studies.

**Key Features**

* Numerical Ability Assessment
* Stroop Test for attention and inhibitory control
* Mental Rotation Task for spatial cognition
* Dataset-based Z-score normalization
* Composite Cognitive Score (CCS) generation
* Age-adjusted cognitive evaluation
* Machine learning–based risk classification
* Interactive Streamlit web interface


**Tech Stack:** Python, Streamlit, Pandas, Scikit-learn, Joblib, OpenPyXL
